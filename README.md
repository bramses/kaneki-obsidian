# kaneki obsidian

obsidian companion for [Kaneki](https://github.com/bramses/kaneki)

## usage

open the command pallate (cmd + p), and type in 'kaneki'

### Send to Kaneki

updates kaneki to update mdx file in `blog` repo

### Open in Chrome

kaneki will open a brower to `blog/slug`
